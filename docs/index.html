05 · Tokenomics

How the Emission Works

CLAWING uses a dual-layer structure — Eras and Epochs — to control token release. Think of Eras as seasons, Epochs as weeks within each season. Here's how the game works.

Era Halving Cycle

1,050,000 blocks ≈ 146 days

An Era is the halving cycle. When a new Era begins, the base mining reward (perBlock) is cut in half. Era 1 starts at 100,000 CLAW per claim; Era 2 drops to 50,000; Era 3 to 25,000, and so on. This is modeled after Bitcoin's halving — the earlier you mine, the more you earn.

Each Era contains 21 Epochs. There are 24 Eras total.

Epoch Weekly Cycle

50,000 blocks ≈ 7 days

An Epoch is the emission cycle — roughly one week. Each Epoch has a hard cap on how many tokens can be mined. In Era 1, that cap is 5 billion CLAW per Epoch. When the cap is reached, mining pauses until the next Epoch begins (next week).

This prevents any single week from being over-mined and ensures a steady weekly rhythm.

Era 1 — 21 Epochs × 5B cap each = 105B total
