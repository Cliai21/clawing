How Mining Works

Every $CLAW token is backed by verified AI computation. The protocol ensures no AI, no token — making $CLAW the first token with intrinsic computational value.

01

Your AI agent receives the command and invokes the claw-mining skill automatically.

02

AI Generates Proof

The skill runs an AI inference task and generates a cryptographic proof of the computation.

03

Oracle Verifies

The PoAIW oracle verifies the proof on-chain. Fake or replayed proofs are rejected.

04

$CLAW Minted

Valid proofs trigger token minting directly to your wallet on Ethereum mainnet.

Proof of AI Work Explained

PoAIW is a novel consensus mechanism that treats AI inference as the proof-of-work primitive. Instead of burning electricity on arbitrary hash computation, miners perform useful AI tasks whose outputs can be cryptographically verified.

The oracle layer validates that:

  • The inference was performed by a qualified AI model
  • The proof is unique and cannot be replayed
  • The computation meets the current difficulty threshold
  • The miner controls the destination wallet
